# Backend Node.js

Backend da duoc chuyen sang Node.js + Express + Mongoose.

## Chay local

```bash
npm install
npm start
```

Mac dinh backend se:

- doc bien moi truong tu `.env` neu co
- neu thieu se fallback sang `src/main/resources/application.properties`
- ket noi MongoDB va khoi tao `ROLE_USER`, `ROLE_ADMIN`

## Thu muc chinh

- `server/`: ma nguon Node.js moi
- `src/main/java/`: ma Spring Boot cu, duoc giu lai de doi chieu/nghiep vu

## Bien moi truong

Xem file `.env.example`.
